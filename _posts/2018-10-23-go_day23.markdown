### io.Reader


```

package main

import (
	"fmt"
	"io"
	"strings"
)

func main() {
	reader := strings.NewReader("ab cde fghij k l  m")
	s := make([]byte, 5)
	for {
		n, err := reader.Read(s)
		if err != nil {
			if err == io.EOF {
				break
			}
			break
		}
		str := string(s[:n])
		fmt.Println(str)
	}
}

```

輸出結果，每次讀取 5 個 byte

```
ab cd
e fgh
ij k
l  m
```
